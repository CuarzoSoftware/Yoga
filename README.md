## Fedora

Install from [cuarzo/software](https://copr.fedorainfracloud.org/coprs/cuarzo/software/) COPR:

```bash
$ sudo dnf copr enable cuarzo/software
$ sudo dnf install cuarzo-yoga-devel
```
